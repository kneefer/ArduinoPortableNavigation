# SMiW-project
Portable navigation (Arduino shield) based on:
- Arduino (ATmega 328)
- GPS Ublox NEO6MV
- Magnetometer HM5883L
- LCD PCD8544
- SD card

Images and schematics available here: https://github.com/kneefer/SMiW-project/tree/master/Images
